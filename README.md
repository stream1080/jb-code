# jb-code

自定义生成JetBrains Code

## Usage

1、安装依赖，第一次运行脚本后会自动生成密钥和证书

```shell
pip install -r requirements.txt
python ca.py
python key.py
```



2、复制 power_conf:

```text
EQUAL,259190816492***,65537,860106576952***->318722192814***
```

3、 将打印的激活码复制到IDE激活即可
code
```text
JETBRAINS-eyJsaWNlbnNlSWQiOiJKRVRCUkFJTlM***
```
